# ASD预测分析系统

这是一个基于机器学习的相邻节段病变(ASD)预测系统，使用自动分段线性回归(APLR)模型和锚点解释技术。

## 功能特点

- 🎯 高精度ASD风险预测
- 📊 研究结果可视化展示
- ⚓ 锚点规则解释
- 🔮 新样本预测
- 💡 个性化医学建议

## 技术栈

- Streamlit - Web应用框架
- APLR - 可解释机器学习模型
- Anchors - 解释性AI技术
- Plotly - 数据可视化
